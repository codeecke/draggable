# How to use

## draggable

Nachdem Du die main.js aus dem dist-Ordner eingebunden hast, kannst Du jedes Element dragbar machen indem Du ihm die klasse `draggable` gibst.

### Beispiel

```HTML
<img class="draggable" src="...">
<div class="draggable">Hello World</div>
```
